## Prompt（英文版，适用于 Gemini 3 Image Pro）

```
A warm, film-grain styled flat-lay photograph of the board game "Decrypto" on a natural wood table.

- The retro-futuristic game box (white with red-blue stripe, resembling a vintage decoding machine with dials, knobs, and a round red signal window) sits at a slight angle in the upper-left area

Visual style:
- Japanese zakka / film photography aesthetic
- Warm brown and cream color palette with the game's red-blue accents as visual focal points
- Slight film grain texture overlay, slightly faded warm tones
- Soft depth of field, background gently blurred

Title overlay area:
- Reserve clean space in the lower-right quadrant for Chinese text overlay
- The text area should have a semi-transparent warm cream wash

Color palette: warm brown (#8B6914), cream (#FFF8DC), with game's original red (#C41E3A) and navy blue (#1B2A4A) as accents

Mood: cozy, inviting, "come sit down and play with us" feeling

Aspect ratio: 3:4 (768x1024 pixels)
Layout: vertical composition, game box as hero element in upper portion, accessories flowing naturally around it
No watermarks, no text in image, no UI elements
```

## 设计说明

### Organism Level（基础场景）
- **主题**：截码战桌游产品展示 — 日系手作风
- **核心传达**：复古解码机美学 + 温暖聚会感，让人想把这盒游戏带回家
- **调性**：温暖、文艺、生活感、治愈

### Molecule Level（细节细化）
- **关键信息元素**：游戏盒（解码机造型）、红色词语展示板、沙漏、软盘Token、咖啡干花场景道具
- **信息层级**：游戏盒（主角） → 配件散落（次要） → 生活场景道具（氛围）
- **图表类型**：产品展示 / 场景摄影
- **图形元素**：实物产品、自然光影、胶片质感

### Atomic Level（精细细节）
- **色彩编码**：暖棕米色为基底，游戏原色红蓝为视觉锚点
- **标签文字**：无图内文字，预留文字叠加区域
- **布局**：游戏盒左上主体，配件自然流淌，右下留白
- **字体**：N/A（文字后期叠加）

### Parameters Level（技术参数）
- **图像比例**：3:4（768x1024）
- **色彩方案**：暖棕 #8B6914 + 米白 #FFF8DC + 红 #C41E3A + 藏蓝 #1B2A4A
- **渲染风格**：日系胶片摄影风
- **输出格式**：小红书封面图
