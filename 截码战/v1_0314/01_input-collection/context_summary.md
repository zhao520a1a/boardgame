## 规则摘要

### 主题背景
扮演特工，在敌方监听下安全传递加密信息

### 获胜条件
- **立即获胜**：获得2个拦截指示物
- **立即失败**：获得2个错误指示物
- **平局处理**：计分比较 → 词语猜测

### 核心流程
1. **分队入座**：两队面对面，中间用挡板隔开
2. **选定加密者**：每回合轮换
3. **抽取密码卡**：3个数字密码（如4-2-1）
4. **编写提示词**：为每个数字写一个提示词
5. **猜测环节**：
   - 对方先尝试拦截猜测
   - 己方队友再猜测
6. **回合判定**：检查胜利条件

### 关键行动
- **加密者**：根据密码卡数字，描述对应词语
- **破译者**：根据提示词反推密码
- **拦截者**：记录对方提示词，尝试破解

### 特殊规则
- 提示必须与词语**含义本身**相关联
- 禁止：谐音、位置暗示、数字暗示、字数笔画、直接念词、外语翻译
- 信息积累效应：越到后期，对方越容易推断
