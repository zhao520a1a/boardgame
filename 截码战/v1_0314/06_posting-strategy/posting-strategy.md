## 二、封面 A/B 测试方案

### 测试流程

```
第1步：首发封面B（情绪冲击风格）
       ↓
第2步：发布后2小时观察数据
       ↓
第3步：判断点击率
       ├─ > 5% → 保持当前封面
       └─ < 5% → 48h内换封面重发
       ↓
第4步：备选顺序 B → A → C
```

### 封面对比

| 封面 | 风格 | 核心钩子 | 预期表现 | 适用场景 |
|------|------|---------|---------|---------|
| **B（首发）** | 情绪冲击 | 表情感染力 | 高互动率 | 聚会/桌游话题 |
| A | 悬念钩子 | 好奇心缺口 | 高点击率 | 推理话题标签 |
| C | 对比反差 | 直观机制 | 高转化率 | 直接推广场景 |

### 换封面时机

- 点击率 < 3%：立即换封面（选择A）
- 点击率 3-5%：观察24h后决定
- 点击率 > 5%：保持当前封面

## 四、评论区运营

### 置顶评论（发布后立即发）

```
你们聚会最常玩什么桌游？评论区推荐一下，我来安利类似的！👇
```

### 预设回复话术

| 用户提问 | 推荐回复 |
|---------|---------|
| 几个人玩最好玩？ | 4-6人体验最佳！人越多越刺激，但别超过8人哦 |
| 难吗？新手能玩吗？ | 完全能！翻第二张图，3分钟就能学会规则 |
| 和狼人杀比哪个好玩？ | 截码战更轻量，30分钟一局，不需要演技，纯烧脑 |
| 哪里买？ | 某宝搜「截码战」就有，正版认准Go Kids |
| 适合情侣吗？ | 适合！但需要拉朋友一起组队，4人起步 |

### 互动引导策略

1. **争议制造**：适当抛出话题
   - 「这游戏运气成分大还是策略成分大？」
   - 「你们觉得当加密者难还是破译者难？」

2. **收藏引导**：正文中已埋入
   - 段5：「翻到最后一张图看技巧」
   - 结尾：「先收藏，聚会的时候用得上」

3. **场景延伸**：回复中植入更多场景
   - 「团建也很适合！5分钟消灭尴尬」
