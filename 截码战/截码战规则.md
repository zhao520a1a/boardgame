# 截码战 (Decrypto)

|------|------|

| 游戏类型 | 猜词 / 团队对抗 / 推理 |
| 玩家人数 | 4~8人 |

## SETUP | 游戏准备

1. **分队入座**：玩家分成两队，面对面坐在桌子两侧，中间用挡板隔开
2. **抽取题目卡**：每队随机抽取4张题目卡，插入己方挡板内，**注意：双方不能看到对方的题目卡**
3. **准备密码卡**：将白队密码卡和黑队密码卡分别洗混，放在一旁备用
4. **分发记录纸**：每队拿取一张记录纸

---

## TURN_FLOW | 回合流程

每个大回合按以下顺序进行：

### 步骤1：选定加密者

- 每队各选一名玩家担任本回合的加密者，加密者每回合轮换
- 加密者负责描述密码，其他玩家为破译者负责猜测

### 步骤2：抽取密码卡

- 两队加密者同时各自抽取一张己方颜色的密码卡
- 密码卡上有3个数字（如4-2-1），**仅加密者可查看**

### 步骤3：编写提示词（双方同时进行）

- 加密者根据密码卡上的3个数字，查看己方挡板上的对应词语
- 为每个数字编写一个提示词/句，描述对应的词语
- 将3个提示词按顺序填写到记录纸上

> **示例**：密码是4-2-1，对应词语是"意志"、"教练"、"莫斯科"
> - 提示可以是：红旗、运动、桌游

### 步骤4：猜测环节（白队先开始）

1. **白队加密者**向队友念出3个提示词
2. **黑队**将听到的提示词记录在自己的记录纸上
3. **黑队先猜测**：根据提示词尝试猜测白队的3个数字密码
4. **公布结果**：
   - 黑队猜对 → 黑队获得1个**拦截指示物**
   - 黑队猜错 → 轮到**白队队友猜测**
5. **白队队友猜测**：根据提示词猜测己方密码
   - 猜对 → 无事发生
   - 猜错 → 白队获得1个**错误指示物**
6. **公布正确答案**：无论猜测结果如何，白队加密者公布正确的3个数字，双方记录

### 步骤5：黑队回合

- 重复步骤4的流程，由**黑队加密者**念提示词
- **白队**先尝试拦截猜测
- 然后**黑队队友**进行猜测

### 步骤6：回合结束判定

- 检查是否有队伍达成胜利条件（见GAME_END）
- 若未结束，双方加密者将密码卡放回牌库，选定下一位加密者，开始新回合

## SPECIAL_RULES | 特殊规则

### 提示词限制

编写提示词时必须遵守以下规则：

| 禁止行为 | 说明 |
|----------|------|
| 谐音提示 | 不能使用与目标词谐音的字词 |
| 位置暗示 | 不能暗示词语在挡板上的位置（如"左上角那个词"） |
| 数字暗示 | 不能暗示密码数字本身 |
| 字数/笔画 | 不能通过目标词的字数或笔画数来提示 |
| 直接念词 | 不能念出目标词本身或其组成部分 |
| 外语翻译 | 不能使用其他语言翻译目标词 |

**正确做法**：提示必须与词语的**含义本身**相关联。

### 公开信息原则

- 提示必须基于公开信息，不能使用只有特定几个人知道的私人信息
- 例如：不能用"我们上次聚餐的地方"作为提示

## GAME_END | 游戏终局

### 终局触发条件

满足以下任一条件时游戏立即结束：

| 条件 | 结果 |
|------|------|
| 一队获得2个拦截指示物 | 该队**获胜** |
| 一队获得2个错误指示物 | 该队**失败**，对方获胜 |
| 进行8轮后仍未分出胜负 | 进入平局处理 |

### 平局处理

当8轮结束后未分胜负，或出现以下情况时：
- 双方同时获得2个拦截和2个错误指示物
- 双方同时获得2个错误指示物
- 双方同时获得2个拦截指示物

**平局解决方式**（按顺序）：

1. **计分比较**：拦截指示物+1分，错误指示物-1分，分数高者获胜
2. **词语猜测**：若仍平局，每队猜测对方的4个题目词语，猜对多的队伍获胜

## STRATEGY | 策略建议

### 提示词编写技巧

- **避免过于直白**：如描述"奶牛"时用"白色""动物""牛奶"会太容易被对方猜到
- **选择抽象关联**：如用"铁桶"描述"奶牛"（挤奶需要桶），队友能懂但对方难猜
- **考虑历史信息**：随着轮次增加，对方会积累你方词语的线索，要注意变化描述角度
- **误导对手**：可以给出与其他词语都有关、但只有队友能分辨的提示

### 拦截破解技巧

- 利用记录纸上的历史提示，归纳每个数字对应词语的特征
- 观察对方加密者的描述习惯，寻找规律
- 随着信息积累，后期拦截成功率会提高

## VARIANTS | 变体规则

### 私人信息规则（可选）

如果是朋友之间私下游玩，可以考虑放宽"公开信息原则"，允许使用只有队友能理解的私人暗号。这会增加游戏趣味性，但会降低对战的公平性。
