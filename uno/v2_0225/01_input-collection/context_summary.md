## 规则摘要

### 主题背景
经典聚会卡牌游戏，玩家通过出牌策略和功能牌互动制造欢乐氛围。

### 获胜条件
- **单局**：第一个出完所有手牌的玩家获胜
- **整局**：累计达到500分的玩家获得最终胜利

### 核心流程
1. **发牌**：每人7张手牌，翻开一张作为起始牌
2. **出牌**：打出与弃牌区顶部**颜色/数字/符号**相同的牌
3. **摸牌**：无法出牌时摸一张，可选择出或不出
4. **喊UNO**：手牌剩最后一张时必须喊"UNO"，否则罚摸2张

### 关键卡牌

| 卡牌类型 | 效果 |
|---------|------|
| 数字牌(0-9) | 按颜色/数字匹配出牌 |
| 转向牌 | 改变出牌方向 |
| 禁止牌 | 跳过下家 |
| +2牌 | 下家摸2张并跳过 |
| 万能牌 | 随时出，指定颜色 |
| +4万能牌 | 下家摸4张，可被质疑 |

### 特殊规则
- **+4质疑**：打出+4需手中无同色牌，下家可质疑验证
- **双人规则**：转向牌视为跳过牌
- **牌堆耗尽**：弃牌区重洗作为新摸牌区
