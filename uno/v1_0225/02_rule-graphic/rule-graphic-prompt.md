# UNO 规则图生成提示词

## 图像规格
- 尺寸：768x1024 像素（小红书3:4比例）
- 风格：Notion风格信息图 / 扁平化设计
- 配色：UNO经典四色（红#FF5555、黄#FFAA00、蓝#5555FF、绿#55AA55）+ 深色背景

## 内容结构（从上到下）

### 顶部区域 - 标题
- 大标题：UNO 优诺
- 副标题：3分钟上手经典聚会卡牌
- 游戏标签：2-10人 | 15-30分钟 | 7岁+

### 游戏目标区域
- 图标：🏆
- 文字：第一个出完手牌的玩家获胜
- 补充：累计500分赢得整局

### 出牌规则区域（核心）
- 标题：怎么出牌？
- 条件列表：
  - ✅ 颜色相同
  - ✅ 数字相同  
  - ✅ 符号相同
  - ✅ 万能牌（随时可出）
- 无法出牌时：摸1张，符合可立即出

### 卡牌类型区域（4列网格）
| 数字牌 | 功能牌 | 万能牌 | 特殊 |
|--------|--------|--------|------|
| 0-9四色 | 转向↩️ | 普通万能 | +4质疑 |
| 76张 | 禁止🚫 | 改颜色 | 心理战 |
| 基础出牌 | +2➕ | +4万能 | 策略深度 |
| | 各8张 | 各4张 | |

### 关键机制区域
1. **UNO喊牌** 🗣️
   - 剩1张时必须喊"UNO"
   - 忘记罚摸2张

2. **+4质疑** ❓
   - 下家可质疑
   - 质疑成功：你摸4张
   - 质疑失败：下家摸6张

### 快速开始区域
1. 每人发7张牌
2. 翻开1张数字牌开始
3. 顺时针轮流出牌
4. 先出完者胜

### 底部 - 记忆口诀
"颜色数字对得上，万能牌随时亮，剩一张喊UNO，聚会欢乐无限长！"

## 设计要点
- 使用UNO经典卡牌视觉元素
- 图标简洁清晰
- 信息层级分明
- 留白充足，不拥挤
