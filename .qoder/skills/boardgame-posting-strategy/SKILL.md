# 发帖策略

> 职责：提供发布策略建议。

## 输入要求

从上下文摘要及前序阶段获取：
- 游戏名称
- 已产出的封面（3 张）
- 已产出的标题和正文
- 推广目标

## 一、发布策略

### 发布时间

| 时段 | 适用场景 | 原因 |
|------|---------|------|
| 周五 18:00-20:00 | 聚会类桌游 | 用户在规划周末活动 |
| 周日 21:00 | 通用 | 用户在规划下周娱乐 |
| 节前 3-5 天 | 节日相关 | 决策窗口期 |
| 工作日 12:00-13:00 | 轻量级桌游 | 午休浏览高峰 |

### 封面 A/B 测试方案

```
1. 首发：选择 B 线（高点击率）封面
2. 观察：发布后 2 小时查看点击率
3. 判断：
   - 点击率 > 5% → 保持当前封面
   - 点击率 < 5% → 48h 内换封面重发
4. 备选：按 B → A → C 顺序尝试
```

### 评论区运营

| 策略 | 执行 |
|------|------|
| 置顶互动 | 发布后立即评论：「你们聚会都玩什么桌游？评论区推荐一下！」 |
| 场景问答 | 「几个人玩最好玩？」→ 回复中植入更多场景 |
| 收藏引导 | 正文中提醒「翻到最后一张看技巧」→ 驱动滑动和收藏 |
| 争议制造 | 适当抛出争议话题：「这游戏运气成分大还是策略成分大？」 |

### 收藏引导策略

在内容中埋入收藏钩子的关键位置：
- 正文段 5：「翻到最后一张图看技巧」
- P5 技巧页：标题用「老玩家偷偷在用」「99% 的人不知道」
- 结尾：「先收藏 聚会的时候用得上」

## 完整输出模板

```markdown
## 发帖策略

**推荐发布时间**：[具体时间]
**首发封面**：[封面编号及风格线]
**置顶评论**：[互动话术]

## 运营 Checklist

- [ ] 发布后立即置顶互动评论
- [ ] 2h 后查看点击率数据
- [ ] 48h 内评估是否需要换封面
- [ ] 回复前 10 条评论（提升互动率）
```
