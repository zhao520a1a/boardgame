# 文案撰写

> 职责：产出完整的笔记文案（标题 + 正文 + Hashtag + 热点适配标题），匹配平台调性。

## 输入要求

从上下文摘要及前序阶段获取：
- 游戏名称、核心机制、人数/时长
- 推广目标、风格线
- 热点节点
- 规则摘要中的核心刺激点

## 一、标题（14-20 字）

### 4 种标题公式

| 类型 | 公式 | 示例 | 效果定位 |
|------|------|------|---------|
| 数字型 | 数字 + 核心卖点 + 场景 | 1至9人所有人数下，最佳桌游推荐（2025版） | **最稳定**：平均互动最高，适合日常发布 |
| 悬念型 | 悬念事件 + 意外结果 | 好上头😍爱玩财商桌游的可以闭眼冲‼️ | **爆发力强**：单篇天花板最高，适合冲爆款 |
| 共鸣型 | 痛点场景 + 解决方案 | 分享3️⃣款我妹玩到不想走的聚会卡牌游戏 | **共情切入**：第一人称叙事引发讨论 |
| 反转型 | 偏见 + 打脸结果 | 我妈说幼稚 结果她连开三局不停 | **认知差异**：需要强反差支撑，慎用 |

### 公式选择策略

根据发布目标选择公式优先级：

| 发布目标 | 首选公式 | 备选公式 | 原因 |
|---------|---------|---------|------|
| 稳定涨粉 | 数字型 | 共鸣型 | 数字型互动最稳定，收藏率高 |
| 冲击爆款 | 悬念型 | 数字型 | 悬念型单篇爆发力最强 |
| 引发讨论 | 共鸣型 | 反转型 | 第一人称+家庭场景易引共鸣评论 |
| 差异化测试 | 反转型 | 悬念型 | 适合小号测试认知反差内容 |

### 标题撰写规则

**基础规范**
- 字数：**14-20 字**（高互动笔记平均 17.3 字，信息量充足的标题点击率更高）
- 每次产出 4 个标题（每种公式 1 个），供用户选择
- 标题中必须能感受到游戏特色，但不一定要包含游戏名
- 优先用具体场景而非抽象描述
- 有节奏感：前半句制造张力，后半句给结果/反转

**高互动标题必备要素**（至少满足 2 项）

| 要素 | 说明 | 示例 |
|------|------|------|
| 具体数字 | 含数字标题互动率提升 3 倍+ | "6款""TOP1""17款" |
| 人数信息 | 直接匹配用户搜索习惯 | "2人""3-9人""全家" |
| 场景词 | 强场景关联提升点击 | "聚会""假期""约会" |
| 强情绪词 | 激发好奇心和从众心理 | "好上头""闭眼冲""停不下来" |

**高价值关键词库**（优先嵌入）

| 关键词 | 互动效果 | 适用场景 |
|--------|---------|---------|
| 推荐 | ⭐⭐⭐⭐⭐ | 直接表明内容属性，通用 |
| 排名/TOP | ⭐⭐⭐⭐⭐ | 建立权威感，适合盘点类 |
| 聚会 | ⭐⭐⭐⭐ | 强场景关联，社交属性 |
| 必备/必囤 | ⭐⭐⭐⭐ | 制造紧迫感，节日节点 |
| 盘点 | ⭐⭐⭐ | 权威感+信息量，适合合集 |

### 标题优化检查清单

发布前逐项检查：

- [ ] 字数在 14-20 字区间
- [ ] 含有具体数字（款数/人数/排名）
- [ ] 含有场景词或人数信息
- [ ] 前半句有张力，后半句有结果
- [ ] 避免直述型标题（如"XX桌游体验"）

### 跨平台适配（可选）

同一套内容可微调后同步其他平台：

| 平台 | 调整项 |
|------|--------|
| 豆瓣 | 标题去掉夸张感、正文增加理性分析、减少 emoji |
| B站专栏 | 标题加「测评」关键词、正文增加对比分析、加视频引导 |
| 微博 | 压缩正文到 140 字内、强化话题标签 |


## 二、正文（6 段结构）

### 段落模板

```
【段1：情绪开场】1-2句，制造共鸣或好奇
- 个人经历切入 / 夸张场景 / 热点联动
- 例："上周家庭聚会差点吵翻天 直到我掏出这盒牌..."

【段2：游戏多简单】引导翻P2
- 3行内讲清核心玩法
- 明确引导：「翻第二张图看教程 真的3步就学会」
- 例："规则简单到爆！翻牌→比大小→决定要不要继续翻 就这么简单"

【段3：游戏多好玩】引导翻P3
- 核心刺激点 1-2 个
- 制造「想玩」冲动
- 例："但是！简单不等于无聊！每次翻牌都是心跳赌局 翻到重复直接爆炸出局"

【段4：适合什么场景】列表式
- 3-5 个使用场景，短句+emoji
- 例：
  🏠 宅家无聊 → 开一局比刷剧带劲
  🎉 朋友聚会 → 笑到肚子疼
  👨‍👩‍👧‍👦 家庭聚餐 → 老少通吃
  💑 约会 → 比看电影刺激
  🏢 团建破冰 → 5分钟消灭尴尬

【段5：进阶钩子】引导翻P5
- 暗示有隐藏技巧/进阶玩法
- 驱动完整滑动和收藏
- 例："翻到最后一张图 有老玩家偷偷在用的3个技巧 看完胜率翻倍"

【段6：行动号召】
- 购买引导 / 互动引导 / 收藏引导
- 例："快过年了赶紧囤一盒！评论区告诉我你们聚会都玩什么？"
```

### 文案语气规范

| 要求 | 说明 |
|------|------|
| 口语化 | 像跟朋友聊天，不是写说明书 |
| 碎片短句 | 每句不超过 20 字，多用断句 |
| 节奏感 | 「！」和 emoji 制造阅读节奏 |
| 自嘲/夸张 | 增加趣味和共鸣 |
| 禁止项 | 书面语、说明书语气、长段落、「本游戏」「该产品」等 |

## 三、Hashtag 策略（10-12 个）

### 分层结构

| 层级 | 数量 | 功能 | 示例 |
|------|------|------|------|
| 垂类标签 | 3-4 个 | 精准触达桌游圈 | #桌游推荐 #桌游安利 #卡牌游戏 #双人桌游 |
| 泛流量标签 | 3-4 个 | 扩大曝光池 | #聚会玩什么 #朋友聚会 #过年回家 |
| 情绪标签 | 2-3 个 | 共鸣与传播 | #上头桌游 #聚会神器 #治愈系 |
| 游戏专属 | 1-2 个 | 品牌沉淀 | #Flip7 #翻转七 |

### 标签选择规则

- 泛流量标签根据热点节点动态调整
- 情绪标签匹配游戏核心体验
- 游戏专属标签包含中英文名

## 四、热点适配标题库

### 节点关键词替换表

同一套内容，替换标题关键词即可适配不同时间节点：

| 节点 | 替换关键词 | 场景氛围 |
|------|----------|---------|
| 春节 | 过年、除夕、亲戚、年夜饭、春晚 | 家庭团聚 |
| 五一/国庆 | 假期、出游、民宿、露营 | 朋友出行 |
| 周末 | 周末、宅家、下午茶、约饭 | 日常休闲 |
| 团建 | 公司、团建、破冰、同事 | 职场社交 |
| 情侣 | 约会、两个人、情人节、宅家 | 二人世界 |
| 暑假 | 暑假、同学、毕业旅行 | 学生社交 |
| 圣诞/跨年 | 圣诞、跨年、倒计时、派对 | 节日派对 |

### 适配标题生成模板

针对每个节点，套用 4 种标题公式生成适配版本（注意保持 14-20 字）：

| 公式 | 春节版（示例） | 周末版（示例） | 团建版（示例） |
|------|--------|--------|--------|
| 数字型 | 春节必囤桌游TOP1！3步学会全家一起玩到停不下来 | 周末宅家神器TOP1！比刷剧带劲全家都上头 | 团建破冰神器TOP1！5分钟消灭尴尬全组笑疯 |
| 悬念型 | 翻到第7张全家尖叫😍除夕夜最上头的名场面 | 宅家翻牌翻到尖叫！周末玩这个太刺激了 | 团建翻牌翻到老板尖叫！全组笑疯停不下来 |
| 共鸣型 | 亲戚问工资怎么办？掏出这盒桌游直接转移话题 | 周末不知道干嘛？这游戏一开就停不下来 | 分享3款让团建不尴尬的桌游！全组抢着加班玩 |
| 反转型 | 我妈说桌游幼稚？结果她自己连开三局不肯停 | 本来约朋友吃饭聊天 结果全程都在玩这个 | 老板说团建别玩游戏 结果他自己玩得最嗨 |

> **模板使用说明**：以上为参考示例，实际创作时根据具体游戏特点调整，确保标题含有「数字/人数/场景词」中的至少 2 项

### 热点适配执行流程

1. 确认当前热点节点（从输入获取 / 根据当前日期自动判断）
2. 用替换表生成 4 个节点适配标题（优先数字型）
3. 调整正文段 1 的情绪开场匹配节点
4. 调整 Hashtag 中的泛流量标签匹配节点
5. 用「标题优化检查清单」逐项验证

## 五、标题创作避坑指南

基于低互动内容特征总结，创作时应避免：

| 避坑项 | 问题描述 | 反面案例 | 正确做法 |
|--------|---------|---------|---------|
| 直述型标题 | 无信息增量，无张力 | "近期桌游记录""XX桌游开箱" | 加入数字/场景/情绪词制造张力 |
| 标题过短 | 信息量不足，点击欲低 | "清明桌游"（6字） | 扩展至 14-20 字，补充场景和数字 |
| 纯日记流水 | 缺乏实用价值 | "今天玩了XX，挺好玩的" | 转为推荐型："玩了5次还想玩的桌游" |
| 冷门节日强蹭 | 节日与桌游关联弱 | "清明节桌游规则" | 聚焦通用场景："3天小长假必玩桌游" |
| 无数字无场景 | 缺乏具体性 | "好玩的桌游推荐" | "聚会必玩的6款桌游推荐" |

## 执行流程

1. 根据上下文摘要，确定发布目标（稳定涨粉/冲爆款/引发讨论）
2. 根据目标选择首选公式，产出 4 个候选标题
3. 用「标题优化检查清单」验证每个标题
4. 用户选定标题后，撰写 6 段正文
5. 组装 10-12 个 Hashtag
6. 根据热点节点生成 4 个适配标题
7. 完整输出：标题 + 正文 + Hashtag + 热点适配标题
