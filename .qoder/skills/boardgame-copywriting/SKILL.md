# 文案撰写

> 职责：产出完整的笔记文案（标题 + 正文 + Hashtag + 热点适配标题），匹配平台调性。

## 输入要求

从上下文摘要及前序阶段获取：
- 游戏名称、核心机制、人数/时长
- 推广目标、风格线
- 热点节点
- 规则摘要中的核心刺激点

## 一、标题（10-15 字）

### 4 种标题公式

| 类型 | 公式 | 示例 |
|------|------|------|
| 悬念型 | 悬念事件 + 意外结果 | 翻到第7张全家尖叫 过年名场面 |
| 数字型 | 数字 + 核心卖点 + 场景 | 春节必囤桌游TOP1 三步就学会 |
| 共鸣型 | 痛点场景 + 解决方案 | 亲戚问工资？开一局直接转移话题 |
| 反转型 | 偏见 + 打脸结果 | 我妈说幼稚 结果她连开三局不停 |

### 标题撰写规则

- 字数：10-15 字（含空格）
- 每次产出 4 个标题（每种公式 1 个），供用户选择
- 标题中必须能感受到游戏特色，但不一定要包含游戏名
- 优先用具体场景而非抽象描述
- 有节奏感：前半句制造张力，后半句给结果/反转

### 跨平台适配（可选）

同一套内容可微调后同步其他平台：

| 平台 | 调整项 |
|------|--------|
| 豆瓣 | 标题去掉夸张感、正文增加理性分析、减少 emoji |
| B站专栏 | 标题加「测评」关键词、正文增加对比分析、加视频引导 |
| 微博 | 压缩正文到 140 字内、强化话题标签 |


## 二、正文（6 段结构）

### 段落模板

```
【段1：情绪开场】1-2句，制造共鸣或好奇
- 个人经历切入 / 夸张场景 / 热点联动
- 例："上周家庭聚会差点吵翻天 直到我掏出这盒牌..."

【段2：游戏多简单】引导翻P2
- 3行内讲清核心玩法
- 明确引导：「翻第二张图看教程 真的3步就学会」
- 例："规则简单到爆！翻牌→比大小→决定要不要继续翻 就这么简单"

【段3：游戏多好玩】引导翻P3
- 核心刺激点 1-2 个
- 制造「想玩」冲动
- 例："但是！简单不等于无聊！每次翻牌都是心跳赌局 翻到重复直接爆炸出局"

【段4：适合什么场景】列表式
- 3-5 个使用场景，短句+emoji
- 例：
  🏠 宅家无聊 → 开一局比刷剧带劲
  🎉 朋友聚会 → 笑到肚子疼
  👨‍👩‍👧‍👦 家庭聚餐 → 老少通吃
  💑 约会 → 比看电影刺激
  🏢 团建破冰 → 5分钟消灭尴尬

【段5：进阶钩子】引导翻P5
- 暗示有隐藏技巧/进阶玩法
- 驱动完整滑动和收藏
- 例："翻到最后一张图 有老玩家偷偷在用的3个技巧 看完胜率翻倍"

【段6：行动号召】
- 购买引导 / 互动引导 / 收藏引导
- 例："快过年了赶紧囤一盒！评论区告诉我你们聚会都玩什么？"
```

### 文案语气规范

| 要求 | 说明 |
|------|------|
| 口语化 | 像跟朋友聊天，不是写说明书 |
| 碎片短句 | 每句不超过 20 字，多用断句 |
| 节奏感 | 「！」和 emoji 制造阅读节奏 |
| 自嘲/夸张 | 增加趣味和共鸣 |
| 禁止项 | 书面语、说明书语气、长段落、「本游戏」「该产品」等 |

## 三、Hashtag 策略（10-12 个）

### 分层结构

| 层级 | 数量 | 功能 | 示例 |
|------|------|------|------|
| 垂类标签 | 3-4 个 | 精准触达桌游圈 | #桌游推荐 #桌游安利 #卡牌游戏 #双人桌游 |
| 泛流量标签 | 3-4 个 | 扩大曝光池 | #聚会玩什么 #朋友聚会 #过年回家 |
| 情绪标签 | 2-3 个 | 共鸣与传播 | #上头桌游 #聚会神器 #治愈系 |
| 游戏专属 | 1-2 个 | 品牌沉淀 | #Flip7 #翻转七 |

### 标签选择规则

- 泛流量标签根据热点节点动态调整
- 情绪标签匹配游戏核心体验
- 游戏专属标签包含中英文名

## 四、热点适配标题库

### 节点关键词替换表

同一套内容，替换标题关键词即可适配不同时间节点：

| 节点 | 替换关键词 | 场景氛围 |
|------|----------|---------|
| 春节 | 过年、除夕、亲戚、年夜饭、春晚 | 家庭团聚 |
| 五一/国庆 | 假期、出游、民宿、露营 | 朋友出行 |
| 周末 | 周末、宅家、下午茶、约饭 | 日常休闲 |
| 团建 | 公司、团建、破冰、同事 | 职场社交 |
| 情侣 | 约会、两个人、情人节、宅家 | 二人世界 |
| 暑假 | 暑假、同学、毕业旅行 | 学生社交 |
| 圣诞/跨年 | 圣诞、跨年、倒计时、派对 | 节日派对 |

### 适配标题生成模板

针对每个节点，套用 4 种标题公式生成适配版本：

| 公式 | 春节版 | 周末版 | 团建版 |
|------|--------|--------|--------|
| 悬念型 | 翻到第7张全家尖叫 除夕名场面 | 宅家翻牌翻到尖叫 周末太刺激了 | 团建翻牌翻到老板尖叫 全组笑疯 |
| 数字型 | 年夜饭必备桌游TOP1 三步学会 | 周末宅家神器TOP1 比刷剧带劲 | 团建破冰神器TOP1 5分钟消灭尴尬 |
| 共鸣型 | 亲戚问工资？掏出这盒直接转移话题 | 周末不知道干嘛？这游戏一开停不下来 | 团建不尴尬 全组抢着要加班玩 |
| 反转型 | 我妈说幼稚 结果她连开三局不停 | 本来约朋友吃饭 结果全在玩这个 | 老板说团建别玩游戏 结果他玩最嗨 |

### 热点适配执行流程

1. 确认当前热点节点（从输入获取 / 根据当前日期自动判断）
2. 用替换表生成 4 个节点适配标题
3. 调整正文段 1 的情绪开场匹配节点
4. 调整 Hashtag 中的泛流量标签匹配节点

## 执行流程

1. 根据上下文摘要，产出 4 个候选标题
2. 用户选定标题后，撰写 6 段正文
3. 组装 10-12 个 Hashtag
4. 根据热点节点生成 4 个适配标题
5. 完整输出：标题 + 正文 + Hashtag + 热点适配标题
