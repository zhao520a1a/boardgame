# 规则讲解图生成

> 职责：将规则文本转化为高可读性的可视化中文速查图。

## 输入要求

从上下文摘要中获取：
- 游戏名称（中英文）
- 规则摘要（结构化）
- 核心机制关键词
- 风格线（决定视觉调性）

## 教学框架

遵循国际通用规则教学结构，按以下顺序组织信息：

```
1. 主题 → 游戏背景/代入感（一句话）
2. 目标 → 获胜条件（玩家立刻知道要干什么）
3. 回合流程 → 每轮做什么（流程图/编号步骤）
4. 行动细节 → 具体操作选项（图标+短文字）
5. 补充规则 → 特殊情况/常见误区（红色警示框）
```

### 教学黄金三步（核心原则）

| 步骤 | 内容 | 目的 |
|------|------|------|
| 第一步 | 游戏主题 + 机制概述 | 建立代入感 |
| 第二步 | 获胜条件 | 明确目标 |
| 第三步 | 规则本体 | 从核心到细节，循序渐进 |

## 设计规范

### 尺寸与布局

- **比例**：3:4（768x1024）
- **信息密度**：65% 功能设计 + 35% 风格表达
- **留白**：>= 20%，避免信息过载

### 可视化优先

| 优先使用 | 替代方式 |
|---------|---------|
| 图标 + 短标签 | 长段落文字 |
| 流程箭头图 | 编号列表 |
| 对比色块 | 纯文字对比 |
| 红色警示框 | 普通文字提醒 |

### 排版规则

- 字体 >= 10pt，关键信息用高对比度高亮
- 统一色彩系统（与封面风格线一致）
- 统一图标风格（线性/填充/扁平，全局一致）
- 常见误区用红色/橙色警示框标注
- 标题层级清晰：游戏名 > 板块标题 > 内容文字

## 输出格式

### 单页版（默认）

1 张 768x1024 图，包含完整规则速查：

```
┌─────────────────────────┐
│    游戏名（中+英）        │ ← 顶部标题区
├─────────────────────────┤
│  主题 & 获胜条件          │ ← 一句话概述
├─────────────────────────┤
│                         │
│  回合流程（流程图）        │ ← 核心区域，占 50%
│                         │
├─────────────────────────┤
│  行动选项（图标网格）      │ ← 操作细节
├─────────────────────────┤
│  ⚠️ 常见误区             │ ← 红色警示区
└─────────────────────────┘
```

### 分页版（规则复杂时）

3 张 768x1024 图：
- P1：主题 + 获胜条件 + 基础设置
- P2：核心回合流程
- P3：进阶规则 + 常见误区
