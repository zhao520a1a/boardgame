# 小红书内容分析规则

## 适用范围

需要由用户手动指定选择该规则，适用于 `xiaohongshu-boardgame-analyzer` 技能在执行数据采集和内容分析时的标准化规范。

---

## 一、数据采集规范

### 1.1 采集频率限制

| 参数 | 限制值 | 说明 |
|------|--------|------|
| 单次最大采集量 | 50 条 | 超过时分批采集 |
| 滚动间隔 | ≥ 2 秒 | 每次滚动后的等待时间 |
| 详情页访问间隔 | ≥ 2 秒 | 每次打开详情页的间隔 |
| 详情页抽样上限 | 10 条 | 仅对 TOP N 高互动笔记抽样 |

### 1.2 数据字段标准化

**数值单位转换**：

| 原始格式 | 转换结果 | 说明 |
|---------|---------|------|
| 1.2万 | 12000 | 万→×10000 |
| 2万 | 20000 | |
| 856 | 856 | 原值保留 |
| 1.5w | 15000 | 英文缩写处理 |

**时间格式统一**：

| 原始格式 | 转换目标 |
|---------|---------|
| 3天前 | YYYY-MM-DD（向前推算） |
| 1周前 | YYYY-MM-DD |
| 2024-12-15 | 保留原值 |
| 昨天 | YYYY-MM-DD |
| 刚刚 / X小时前 | 当天日期 |

### 1.3 隐私合规

- 仅采集公开可见的笔记数据（标题、互动数、标签等）
- 不采集用户个人信息（头像URL、个人简介、粉丝数等）
- 不存储笔记原文全文，仅存储摘要（前 200 字）
- 采集数据仅用于分析，不做二次传播

### 1.4 数据清洗规则

- **去广告**：移除标记为"赞助""广告""商业合作"的笔记
- **去重**：同一标题 + 同一作者视为重复，仅保留一条
- **异常值过滤**：数据缺失超过 50% 字段的条目标记为无效
- **最低有效数据量**：10 条有效笔记方可进入分析阶段

## 二、标题公式识别标准

### 2.1 四种标题公式定义

| 公式类型 | 识别特征 | 典型信号词 | 示例 |
|---------|---------|-----------|------|
| **数字型** | 含阿拉伯数字 + 推荐/排名类词汇 | TOP、必买、N款、推荐、排行、盘点 | "桌游推荐TOP5 新手闭眼入" |
| **反转型** | 含转折词，前后形成认知反差 | 但是、结果、没想到、打脸、竟然、本以为 | "我妈说幼稚 结果她连开三局" |
| **悬念型** | 含疑问/省略/制造好奇的表达 | ？、...、到底、居然、为什么、真相 | "翻到第7张全家尖叫 过年名场面" |
| **共鸣型** | 含第二人称 + 痛点/场景描述 | 你、我们、是不是、每次、总是 | "亲戚问工资？开一局直接转移话题" |

### 2.2 识别优先级

当一条标题同时符合多种公式时，按以下优先级归类：

```
数字型 → 反转型 → 悬念型 → 共鸣型 → 其他
```

理由：数字型特征最明确（含数字），反转型次之（含明确转折词），悬念型和共鸣型可能重叠较多。

### 2.3 "其他"类型处理

不符合以上四种公式的标题归为"其他"，但仍需标注其风格特征：
- 直述型（直接描述产品/内容）
- 教程型（教你/学会/攻略）
- 清单型（合集/清单/整理）

## 三、互动数据计算方法

### 3.1 综合互动分

```
综合互动分 = 点赞 × 1 + 收藏 × 2 + 评论 × 3
```

**权重依据**：
- 点赞：最轻量的互动行为，权重 1
- 收藏：表示内容"有用"，用户主动保存，权重 2
- 评论：最深度的互动行为，表示强参与，权重 3

### 3.2 高互动阈值

- **高互动**：综合互动分 ≥ 样本 P80 分位值（TOP 20%）
- **中互动**：综合互动分在 P40-P80 之间
- **低互动**：综合互动分 < P40

### 3.3 缺失数据处理

| 缺失字段 | 处理方式 |
|---------|---------|
| 仅有点赞 | 综合互动分 = 点赞 × 1（标注"仅点赞数据"） |
| 缺少收藏 | 综合互动分 = 点赞 × 1 + 评论 × 3 |
| 缺少评论 | 综合互动分 = 点赞 × 1 + 收藏 × 2 |
| 全部缺失 | 标记为无效条目，不参与互动分析 |

## 四、分析维度权重

在生成综合策略建议时，各维度的参考权重：

| 维度 | 权重 | 说明 |
|------|------|------|
| 标题分析 | 30% | 标题是点击率的第一决定因素 |
| 互动数据 | 25% | 反映内容质量和受众匹配度 |
| 标签策略 | 20% | 影响内容分发和曝光范围 |
| 内容结构 | 15% | 影响完播率/滑完率和收藏率 |
| 情感分析 | 10% | 辅助判断内容调性方向 |

---

## 五、小样本分析标注

当有效数据量 < 20 条时，报告中必须标注：

```markdown
> ⚠️ 小样本分析提示：本次分析基于 [N] 条有效数据，样本量偏小，
> 结论仅供参考，建议增加数据量后重新分析以提高置信度。
```

同时，对受样本量影响较大的分析维度（如标题公式分布、情感倾向分布）降低结论的确定性表述，使用"初步观察""可能存在"等措辞。
