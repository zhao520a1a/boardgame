# 短视频脚本创作规范

> 定位：控制"怎么写短视频脚本"——从结构设计到分镜落地的完整方法论。
> 核心公式：**冲突开场 → 好奇驱动 → 反转收尾**

## 适用范围

当任务涉及**短视频脚本创作**时启用本规则，包括但不限于：
- 抖音、快手、视频号等平台的短视频脚本
- 产品种草 / 开箱 / 测评类短视频
- 剧情 / 情景类短片脚本
- 知识科普 / 干货分享类口播脚本
- 任何需要"让人看完、看完想互动"的 15-60 秒视频内容

本规则侧重**脚本结构与节奏控制**，具体的文字风格参见 `writing-style.md`。

---

## 一、三段式核心结构

短视频脚本必须包含以下三个结构模块，缺一不可：

| 模块 | 时间占比 | 核心目标 | 对应平台指标 |
|------|---------|---------|-------------|
| 冲突开场 | 前 5 秒（约 10-15%） | 抓住注意力，阻止划走 | 3 秒完播率 |
| 好奇驱动 | 中段（约 60-70%） | 维持观看欲望，持续制造"想知道" | 平均播放时长 |
| 反转结局 | 末尾（约 15-20%） | 制造惊喜，触发互动行为 | 完播率、点赞、评论、转发 |

---

## 二、模块一：冲突开场（前 5 秒黄金时间）

### 2.1 设计原则

前 5 秒是短视频的生死线。观众在 1-3 秒内决定是否继续观看，冲突必须**即时、强烈、直觉可感**。

### 2.2 六种冲突类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 认知冲突 | 打破常识，制造"这不可能"的反应 | "这款游戏，输的人反而是赢家" |
| 情境冲突 | 展示强烈反差的场景 | 一群人围着桌子尖叫（画面）→ "他们在玩的，只是一封信" |
| 悬念冲突 | 抛出一个令人好奇的问题或结果 | "这局游戏，我用一张牌淘汰了所有人" |
| 情感冲突 | 呈现人与人之间的矛盾张力 | "我最好的朋友，刚才在游戏里骗了我" |
| 视觉冲突 | 用画面本身制造冲击 | 桌上摆满精美配件的特写 → 突然全部推倒 |
| 数据冲突 | 用反直觉的数字引发关注 | "一个只有16张牌的游戏，凭什么卖了500万套" |

### 2.3 冲突开场模板

```
模板 A（反常识）：「你以为……其实……」
模板 B（结果前置）：「最后的结果是……但一开始……」
模板 C（直接提问）：「你敢信？……竟然……」
模板 D（冲突画面）：[画面描述] + 一句悬念旁白
```

### 2.4 红线

- 禁止使用慢节奏的自我介绍开头（"大家好，我是xxx"）
- 禁止使用与内容无关的标题党式冲突
- 冲突必须与后续内容逻辑相连，不能断裂

## 三、模块二：好奇心驱动（中段内容）

### 3.1 设计原则

中段的目标是让观众**持续产生"然后呢？"的心理**。信息释放节奏是关键——不能一次给完，也不能吊太久。

### 3.2 悬念点设计要求

中段必须设置**至少 2-3 个悬念点**，按以下节奏分布：

| 悬念点 | 位置 | 功能 | 手法 |
|--------|------|------|------|
| 悬念点 1 | 冲突之后（约第 6-10 秒） | 承接冲突，引出故事线 | 给出部分背景，但保留关键信息 |
| 悬念点 2 | 中段核心（约第 10-30 秒） | 推进剧情，制造转折期待 | 抛出新变量、新矛盾或意外进展 |
| 悬念点 3 | 反转前夕（约倒数 5-10 秒） | 把期待拉到最高点 | 营造"即将揭晓"的紧张感 |

### 3.3 信息释放策略

| 策略 | 说明 | 示例 |
|------|------|------|
| 逐步揭露 | 像剥洋葱一样一层层释放信息 | 先说现象 → 再说原因 → 最后说结论 |
| 预期偏转 | 引导观众建立一个预期，然后偏离它 | "你以为他要出这张牌——但他选了另一张" |
| 倒计时感 | 让观众感知到结果即将到来 | "最后一轮，只剩两个人……" |
| 角色代入 | 让观众代入某个角色的处境 | "如果你手里只有这张牌，你怎么选？" |

### 3.4 节奏控制

- 每 5-8 秒必须出现一个新的信息点或情绪变化
- 避免连续超过 8 秒的平铺直叙
- 语速、画面切换、音效三者配合调节节奏

---

## 四、模块三：反转结局（提高完播率）

### 4.1 设计原则

反转是短视频的"钩子"。一个好的反转让观众产生**"没想到！"的惊喜**，并触发点赞、评论、转发甚至二刷行为。

### 4.2 反转的三条铁律

1. **逻辑自洽**：反转必须在意料之外、情理之中。回看时所有铺垫都指向这个结局
2. **情绪落差**：反转前后的情绪必须形成明显对比（紧张→搞笑、期待→意外、悲伤→温暖）
3. **余韵感**：反转之后留 1-2 秒的"空白感"，让观众消化并产生互动冲动

### 4.3 五种反转类型

| 类型 | 机制 | 示例 |
|------|------|------|
| 身份反转 | 揭示角色的真实身份或立场 | 一直在教别人玩的"高手"其实是第一次玩 |
| 结果反转 | 事件结果与预期完全相反 | 稳赢的人最后一轮被翻盘 |
| 视角反转 | 切换叙事视角，呈现新的真相 | 从另一个玩家角度看，之前的操作全是计策 |
| 情感反转 | 从竞争转向温情，或反过来 | 激烈对抗后发现是生日惊喜策划 |
| 认知反转 | 颠覆观众对某件事的理解 | "这些复杂策略其实只需要一条规则" |

### 4.4 反转落地技巧

- 反转点用**画面 + 音效 + 表情**三重强化
- 反转后配合一句金句或点题台词
- 结尾可加一句互动引导（"你猜到了吗？""评论区告诉我你的选择"）

## 五、脚本格式规范

### 5.1 标准分镜脚本模板

每个镜头按以下字段描述：

```
【镜头 #序号】（时间：xx-xx秒）
[结构模块]：冲突开场 / 好奇驱动 / 反转结局
[景别]：特写 / 近景 / 中景 / 全景
[画面描述]：具体的视觉内容
[台词/旁白]："具体的文字内容"
[音效/BGM]：音效类型或音乐情绪提示
[字幕]：屏幕上需要显示的文字（如有）
[运镜]：固定 / 推 / 拉 / 摇 / 移 / 手持晃动（如需指定）
```

### 5.2 格式要求

| 项目 | 要求 |
|------|------|
| 总时长 | 15-60 秒，根据平台和内容类型调整 |
| 台词密度 | 口播类约 3-4 字/秒；剧情类根据节奏灵活调整 |
| 镜头数量 | 15 秒视频约 3-5 个镜头；60 秒视频约 8-15 个镜头 |
| 字幕标注 | 关键信息、金句、互动引导语必须标注字幕 |
| 音效标注 | 冲突点、悬念点、反转点必须标注音效提示 |

## 六、脚本创作流程

### 步骤一：明确脚本定位

```
[主题]    ：_______________（视频要讲什么）
[目标]    ：_______________（让观众感到/知道/做到什么）
[平台]    ：_______________（抖音/快手/视频号/B站等）
[时长]    ：_______________（15秒/30秒/60秒）
[类型]    ：_______________（剧情/口播/开箱/教学/种草等）
```

### 步骤二：设计三段式结构

1. 确定**冲突类型**（从六种中选择最适合的）
2. 规划**悬念点分布**（至少 2-3 个，标注在时间线上）
3. 设计**反转类型**（从五种中选择，确保与铺垫逻辑一致）

### 步骤三：撰写分镜脚本

按标准模板逐镜头填写，确保：
- 每个镜头有明确的视觉和听觉信息
- 台词自然口语化，避免书面语
- 音效和 BGM 情绪与画面匹配

### 步骤四：节奏校验

- 大声朗读台词，用秒表计时，确认时长在目标范围内
- 检查每 5-8 秒是否有新信息点或情绪转折
- 确认冲突在前 5 秒内完成呈现

### 步骤五：自检定稿

通过自检清单后方可定稿（见下方）。

## 七、脚本自检清单

脚本完成后逐项过一遍，**全部通过方可定稿**：

### 结构完整性

- [ ] 前 5 秒有明确的冲突/矛盾设置？
- [ ] 冲突能让人停下滑动、产生"什么情况"的反应？
- [ ] 中段设置了至少 2-3 个悬念点？
- [ ] 每个悬念点之间有信息递进，不是简单重复？
- [ ] 结尾有明确的反转或意外？
- [ ] 反转逻辑自洽，回看时铺垫合理？

### 节奏与时长

- [ ] 总时长在 15-60 秒范围内？
- [ ] 没有超过 8 秒的平铺直叙段落？
- [ ] 信息密度均匀，无明显的"空白"或"堆砌"？

### 制作可行性

- [ ] 每个镜头都有画面描述、台词、音效提示？
- [ ] 台词口语化，朗读顺畅？
- [ ] 视觉元素可实际拍摄或制作？
- [ ] 字幕标注完整，关键信息不遗漏？

### 互动设计

- [ ] 结尾有互动引导（提问/投票/挑战）？
- [ ] 内容有让人想评论或转发的"槽点"或"共鸣点"？

## 八、常见问题与对策

| 问题 | 原因 | 对策 |
|------|------|------|
| 3 秒完播率低 | 开头冲突不够强或太慢 | 用更强烈的冲突类型；把最劲爆的画面/台词放在第 1 秒 |
| 平均播放时长短 | 中段悬念不足，信息一次给完 | 增加悬念点数量；用"预期偏转"制造新的好奇 |
| 完播率低 | 反转缺失或可预测 | 换一种反转类型；增加反转前的铺垫使预期偏离更远 |
| 互动率低 | 缺少互动设计 | 结尾加入互动引导；在反转处设置观众可以讨论的争议点 |
| 台词念不顺 | 书面语太重 | 大声朗读修改，用日常说话的方式重写 |
