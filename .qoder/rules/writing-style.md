# 写作风格与表达规范

> 来源：版本A_精炼版 SKILL 通用化提炼。
> 定位：控制"怎么写"——语气、结构、节奏、自检。

## 适用范围

当任务涉及**为读者产出文字内容**时启用本规则，包括但不限于：
- 社交媒体图文笔记（小红书、公众号等）
- 产品种草 / 推荐文案
- 短篇分享、攻略、测评
- 任何需要"让人想读、读完有获得感"的文字

本规则**不**覆盖写作前的选题和定位流程（参见 `writing-process.md`）。

---

## 一、写作心法（三条铁律）

1. **先当读者，再当作者**
   动笔前问自己——这个标题我会点吗？这段话我会读下去吗？读完我能得到什么？

2. **聊天不说教**
   你在跟朋友安利一个好东西，不是在写说明书。平等分享，禁止居高临下。

3. **凤头猪肚豹尾**
   开头一秒抓住人，中间给足料，结尾干脆利落不拖泥带水。

---

## 二、写作前三个必答题

| 问题 | 必须明确 |
|------|---------|
| 我写给谁看？ | 具体的人（如"桌游小白""职场新人"），不是"所有人" |
| 读者的痛点是什么？ | 他正在为什么事发愁 / 好奇 / 犹豫？ |
| 读完 ta 能得到什么？ | 一个解决方案 / 一种冲动 / 一份可以直接用的东西 |

答不上来，不要开始写。

---

## 三、正文结构口诀

> 像一条溪流——顺着读者的心理流向走，**可以拐弯但不可分叉**。

### 凤头（开头 10-15%）
- 从读者的真实场景切入，制造"对对对，我就是这样"的共鸣
- 或抛一个悬念，让人想知道后面发生了什么
- 3 秒内建立阅读动机

### 猪肚（正文 70-80%）
- 核心内容给足干货，论点 + 案例/数据
- 正反结合，用小标题分层
- 时不时抛个问题给读者，调动参与感
- 每段只讲一个信息点

### 豹尾（结尾 10-15%）
- 一句话行动号召，不啰嗦
- 或总结价值 / 金句收束
- 戛然而止，不拖泥带水

---

## 四、万能写作公式

当缺乏明确思路时，按以下四步展开：

1. **场景化切入**：描述一个读者熟悉的场景，引出痛点
2. **观点输出**：针对痛点提出你的新鲜观点
3. **双面论证**：正面案例 + 反面对比，小标题分层
4. **行动号召**：总结价值，给出具体可执行建议

## 五、语气红绿灯

| 绿灯（鼓励） | 红灯（禁止） |
|-------------|-------------|
| 跟朋友聊天的语气 | 说明书 / 产品介绍语气 |
| 碎片短句，每句不超 20 字 | 长段落、复杂从句 |
| 自嘲、夸张、反问 | "本产品""该方案""值得购买" |
| 分享自己的真实感受 | 单向灌输信息 |
| 用读者能理解的语言 | 堆砌专业术语 |
| 适度提问引发互动 | 全篇无互动点 |

### 接收优先原则

> 以读者接收到的信息为准，而非作者表达的信息为准。

写完每段后自问：一个不了解背景的读者，能准确理解这段话吗？

---

## 六、发布前自检清单

写完后逐项过一遍，全部通过方可定稿：

- [ ] 我作为读者会点这个标题吗？
- [ ] 读完能感受到获得感（知识/启发/共鸣/冲动）吗？
- [ ] 全文一条主线，无旁枝分叉？
- [ ] 有没有说教感？像不像朋友在聊天？
- [ ] 有没有至少一个互动点（提问/投票/留悬念）？
- [ ] 开头 3 秒抓人？
- [ ] 结尾干脆有力，没有废话？
- [ ] 用读者的语言，而非专业术语？

---

## 七、选题底线

写作素材必须包含以下至少一项：

- **态度**：你对某件事/现象的真实看法
- **发现**：你在生活中的观察和独到洞察

不携带态度或发现的文字是信息搬运，不是写作。
