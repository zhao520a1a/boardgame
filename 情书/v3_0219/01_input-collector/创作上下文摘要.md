## 游戏信息

| 字段 | 内容 |
|------|------|
| 游戏名称 | 情书 (Love Letter) |
| 游戏类型 | 卡牌 / 推理 / 淘汰 |
| 玩家人数 | 2~4人（最佳4人） |
| 游戏时长 | 约20分钟 |
| 适合年龄 | 8岁以上 |
| 一句话介绍 | 每人手里只有1张牌，抽1张打1张，用牌的能力猜测和淘汰对手，活到最后或点数最大的人赢 |

## 核心卖点

1. **极简规则**：手里永远只有1张牌，3分钟学会
2. **推理烧脑**：8种角色能力，每张打出的牌都是线索
3. **猜中爽感**：卫兵猜对对手牌直接淘汰，紧张刺激
4. **公主悖论**：点数最大但打出就死，要死死攥住
5. **便携极致**：16张牌一小袋，随时随地开一局
