# 《情书》发帖策略

> 版本：v3_0219 | 创建日期：2026-02-19

## 推荐发布时间

| 时段 | 原因 |
|------|------|
| 周五 18:00-20:00 | 用户规划周末活动高峰 |
| 工作日 12:00-13:00 | 午休浏览高峰，轻量桌游匹配度高 |

## 封面 A/B 测试方案

1. 首发用 B线（悬念钩子）封面 — 暗调翻牌特写，点击率优先
2. 发布 2h 后查看点击率
3. 点击率 < 5% → 48h 内换 A线（烛光夜话）封面重发
4. 备选顺序：B → A → C
