# 情书 Love Letter — 发帖策略

---

## 一、图片排列顺序（推荐方案）

小红书笔记最多放9张图，推荐以下**6张图**排列组合：

| 顺序 | 图片 | 作用 |
|------|------|------|
| P1 | 封面（选一张） | 吸引点击 |
| P2 | 内页1-教学页（3步学会） | 承接正文段2引导「翻第二张图看教程」 |
| P3 | 内页2-角色解析页 | 承接正文段3引导「翻第三张图看角色能力」 |
| P4 | 内页3-场景页 | 展示使用场景，强化种草 |
| P5 | 内页4-技巧页 | 承接正文段5「翻到最后一张图看技巧」，驱动收藏 |
| P6 | 规则速查图 | 完整规则参考，提升收藏价值 |

> 素材图片（桌游盒子.jpg、桌游纸牌.jpg、角色技能.jpg）可替换P4场景页或补充到P6之后，作为实物展示。

|---------|---------|------|

| 时段 | 适用场景 |
|------|---------|
| 周五 18:00-20:00 | 最佳时段，用户在规划周末聚会活动 |
| 工作日 12:00-13:00 | 午休浏览高峰，轻量内容曝光好 |

## 三、封面A/B测试方案

### 测试顺序

```
第1轮（首发）：封面A 文艺调性
   ↓ 匹配涨粉目标，文艺调性账号调性强
   ↓ 发布后观察2小时点击率
   ↓
判断：
  → 点击率 > 5% → 保持不动，持续观察48h
  → 点击率 < 5% → 进入第2轮

第2轮（48h内换封面重发）：封面B 高点击率
   ↓ 强对比大文字，争取更高曝光
   ↓
第3轮（备选）：封面C 人物产品
   ↓ 生活场景感强，适合种草转化
```

### 为什么首发选封面A？

- 推广目标是**涨粉**，文艺调性封面能建立账号风格辨识度
- 情书的浪漫主题与文艺风格高度匹配
- 情人节/七夕节点，用户对「浪漫美学」内容更敏感

## 四、评论区运营建议

### 置顶评论（发布后立即发）

**首选话术（涨粉+互动向）：**
> 你们约会/聚会都玩什么桌游？评论区推荐一下！我再出测评~

**备选话术（争议向，冲互动量）：**
> 灵魂拷问：情书到底是靠推理还是靠运气？我觉得7分推理3分运气，你们呢？

### 评论回复策略

| 评论类型 | 回复策略 | 示例 |
|---------|---------|------|
| 问哪里买 | 引导但不硬推 | 「各大桌游店和网店都有哦 搜"情书桌游"就行 小盒子很便宜的」 |
| 问几人玩好 | 植入更多场景 | 「3-4人最上头！2人约会也很浪漫 有专门的二人规则」 |
| 说不会玩 | 引导看图 | 「翻到第二张图！真的只有3步 10秒学会」 |
| 分享体验 | 热情互动+追问 | 「哈哈哈守卫猜中的那一刻太爽了吧！你们最长连开几局？」 |
| 推荐其他游戏 | 接话+预告 | 「这个我也想试！下期可以出个测评对比 先关注不迷路~」 |

### 运营节奏

- **前2小时**：每条评论都回复，冲初始互动量
- **前24小时**：回复所有评论，维持算法热度
- **48小时后**：挑选高质量评论回复，保持活跃

## 六、情人节/七夕双节点复用方案

同一套内容可以发**两次**（间隔足够长），只需调整：

| 调整项 | 情人节版 | 七夕版 |
|--------|---------|--------|
| 标题 | 情人节最浪漫的事 是用一张牌猜透你的心 | 七夕约会不踩雷 这盒宫廷情书比看电影刺激十倍 |
| 正文段1 | 「情人节不想只吃饭看电影？试试这个...」 | 「七夕约会除了吃饭逛街 还能干嘛？」 |
| Hashtag替换 | #情人节约会 → #七夕约会 #七夕礼物 | 同左反向 |
| 其他内容 | 保持不变 | 保持不变 |
