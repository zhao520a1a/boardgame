## 封面 A/B 测试方案

### 测试顺序

| 优先级 | 封面 | 配合场景 |
|--------|------|---------|
| 1️⃣ 首发 | 封面C（情侣咖啡厅） | 约会热点，信任感强 |
| 2️⃣ 备选 | 封面A（文艺平铺） | 种草向，品质感 |
| 3️⃣ 流量 | 封面B（暗调悬疑） | 涨粉向，配悬念标题 |

### 测试流程

```
1. 首发：封面C + 悬念型标题
2. 观察：发布后 2 小时查看点击率
3. 判断：
   - 点击率 > 5% → 保持当前封面
   - 点击率 < 5% → 48h 内换封面A重发
4. 二次测试：若A效果仍不佳，尝试封面B + 反转型标题
```
