# 《情书》小红书图文笔记 - 创作上下文摘要

## 游戏基本信息

| 项目 | 内容 |
|------|------|
| 游戏名称 | 情书 (Love Letter) |
| 游戏人数 | 2-4人（最佳4人） |
| 游戏时长 | 约20分钟 |
| 适合年龄 | 8岁以上 |
| 游戏类型 | 卡牌 / 推理 / 淘汰 |

## 核心玩法提炼

- **极简手牌**：全程手里只有1张牌，抽1张打1张，决策极度聚焦
- **推理淘汰**：通过打出的牌推理对手手牌，用能力猜测和淘汰对手
- **信息博弈**：每张打出的牌都是公开信息，越到后期推理越精准
- **角色能力**：8种角色各有独特能力，从猜牌到偷看到交换，互相克制
- **多轮竞赛**：每轮很快结束，赢家获得爱心标记，攒够一定数量赢得游戏

## 游戏卖点（种草角度）

1. **极简规则大乐趣**：手里永远只有1张牌，3分钟教会，但策略深度惊人
2. **超快节奏**：一轮可能几分钟就结束，全程高能不拖沓
3. **推理烧脑**：每张打出的牌都是线索，猜对一张牌就能淘汰对手的快感
4. **社交互动爆棚**：猜牌、偷看、交换、比大小，每一步都在和人互动
5. **经典常青**：全球桌游圈经典之作，16张牌撑起无数名局
6. **便携到极致**：一小袋牌随身带，随时随地开一局

## 种草策略要点

- 突出"只有16张牌"的极简感，制造"这么简单居然这么好玩"的反差
- 强调推理猜牌的社交刺激感，用具体场景还原"猜中对手那一刻"
- 利用"情书"这个名字的浪漫联想，适配情侣/约会场景
- 展示"口袋桌游"的便携性，降低购买门槛

## 目标受众

- 桌游入门新手，寻找简单好上手的游戏
- 情侣约会，找有趣的二人互动游戏
- 聚会社交需求者，找快节奏派对游戏
- 轻度策略爱好者，喜欢推理和心理博弈
