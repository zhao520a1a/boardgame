# 《情书》发帖策略

## 发布策略

**推荐发布时间**：周五 18:00-20:00（用户规划周末聚会）或 周日 21:00（规划下周娱乐）
**首发封面**：封面A 文艺调性（烛光情书），契合"情书"浪漫主题，适合种草账号调性
**备选封面**：封面C 人物产品（情侣约会），适配情人节/约会场景节点

### 封面 A/B 测试方案

1. 首发：使用封面A（文艺调性）
2. 观察：发布后 2 小时查看点击率
3. 判断：
   - 点击率 > 5% → 保持当前封面
   - 点击率 < 5% → 48h 内换封面C（情侣约会）重发
4. 备选顺序：A → C → B

### 置顶评论

「你们玩情书被卫兵猜中过几次？我上一局被连猜三轮 差点掀桌 评论区说说你的名场面！」

### 评论区运营

| 策略 | 执行 |
|------|------|
| 置顶互动 | 发布后立即评论上述互动话术 |
| 场景问答 | 回复中植入更多场景：「两个人也超好玩！约会必备」 |
| 收藏引导 | 回复提醒：「记得翻到最后一张看技巧哦」 |
| 争议制造 | 抛出话题：「情书是靠运气还是靠推理？」 |

### 收藏钩子位置

- 正文段5：「翻到最后一张图有老玩家偷偷在用的3个技巧」
- P5技巧页标题：老玩家的3个秘诀
- 结尾：「记得收藏 下次聚会前翻出来复习一下」

- [ ] 回复前 10 条评论（提升互动率）
- [ ] 情人节/约会类热点时段换用封面C重发
