# 内页设计说明

## 整体风格
- 风格线：A线文艺调性（暖金复古烛光感）
- 主色调：暗金暖棕 + 酒红 + 羊皮纸质感
- 与封面A统一视觉语言

## 各页内容

### P2 教学页 —「4步学会情书」
- 复古羊皮纸风格，4步金色编号卡片垂直排列
- 步骤：准备牌堆 → 了解点数 → 抽牌出牌 → 赢得胜利
- 配经典扑克牌和皇冠图标，降低门槛感
- 需后期叠加中文步骤文案

### P3 解析页 —「为什么让人上瘾？」
- 红心Q与黑桃K对决构图，金色光芒碰撞
- 暗红丝绒背景 + 烛台氛围
- 三枚徽章图标代表：推理快感 / 淘汰刺激 / 信息博弈
- 传递猜牌对决的核心张力

### P4 场景页 —「闺蜜局 / 朋友聚会」
- 四人围坐沙发，暖光客厅自然聚会场景
- 真实笑容和惊讶表情，代入感强
- 桌上可见卡牌和标记物，烛光 + 仙女灯串
- 标语建议：「20分钟一局 笑到停不下来」

### P5 技巧页 —「老玩家的3个秘诀」
- 复古卷轴羊皮纸风格，3个编号技巧卡
- Tip 1 OBSERVE：观察已出牌推理对手
- Tip 2 STRATEGIZE：规划出牌顺序和时机
- Tip 3 PROTECT：关键时刻用侍女自保
- 需后期叠加中文技巧文案
