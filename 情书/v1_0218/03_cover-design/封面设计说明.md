# 封面设计说明

## 主封面选定
- **选定方案**：A线 - 文艺调性（烛光情书）
- **视觉风格**：复古暖调，烛光 + 信封蜡封 + 卡牌散铺
- **主色调**：暗金暖棕 + 酒红色

## 三张封面概述

### 封面A - 文艺调性（主封面）
- 子风格：烛光夜话
- 视觉：烛光照亮的暗木桌面，蜡封情书、卡牌散铺、干花点缀
- 标题："情书" + "Love Letter" + "只有16张牌的推理对决"
- 适用：打造文艺桌游号调性，种草安利，长期可用

### 封面B - 高点击率
- 子风格：悬念钩子
- 视觉：红丝绒上的红心Q牌 + 怀表 + 烛台，暗调电影感
- 需后期叠加悬念文案："猜错一张牌 直接出局"
- 适用：冲流量、吸引点击

### 封面C - 人物产品
- 子风格：约会实拍
- 视觉：情侣咖啡馆约会，暖光自然互动，卡牌散落桌面
- 标签："约会神器 猜心大作战"
- 适用：情侣场景种草，社交代入感强

## 内页风格方向
后续内页（P2-P5）将延续 A线文艺调性的视觉语言：暖金色调 + 复古质感 + 手作感排版元素。
