## 🎮 核心机制

**主题背景：** 玩家作为领主派遣追随者占领道路、城市和修道院，争夺南法卡卡颂地区的控制权

**核心机制：**
- 板块放置（Tile Placement）
- 区域控制（Area Control）
- 资源管理（小人数量限制：7个）
- 时机选择（何时占领/何时完成/何时收回）

**游戏组件：**
- 72张地形拼图（道路/城市/草地/修道院）
- 5种颜色小人（每人8个，1个用于记分）
- 1块记分板（0~49分轨道）
- 扩展包：河流拼图12张、修道院长小人

---

## 🔄 游戏流程

### 回合结构

每回合固定4步：

1. **抽拼图** - 从牌堆顶抽1张
2. **放拼图** - 必须边对边相接且元素匹配（道路接道路、城市接城市、草地接草地）
3. **放小人（可选）** - 最多放1个到刚放的拼图上，但不能放在已被占领的区域
4. **立即计分（如有）** - 区域完成立刻结算，小人收回

### 小人角色

- 站在道路上 → **路霸**
- 站在城市里 → **骑士**
- 站在修道院上 → **修士**
- 横躺在草地上 → **农民**（进阶规则）

## 💰 计分规则

### 游戏进行中（完成区域）

| 区域类型 | 计分规则 |
|---------|---------|
| **完成的道路** | 每张拼图 = 1分 |
| **完成的城市** | 每张拼图 = 2分 + 盾牌 = 额外2分 |
| **完成的修道院** | 九宫格 = 9分（包括修道院本身） |

### 游戏结束时（未完成区域）

| 区域类型 | 计分规则 |
|---------|---------|
| **未完成道路** | 每张拼图 = 1分 |
| **未完成城市** | 每张拼图 = 1分 + 盾牌 = 1分 |
| **未完成修道院** | 每张拼图 = 1分 |
| **农民** | 每个完成城市 = 3分 |

## ⚡ 核心策略要点

### 1. 区域共享机制

- **限制：** 已被占领的区域不能再放小人
- **绕过方法：** 在独立区域放人，后用拼图连接两区域
- **共享结算：**
  - 人数相等 → 所有玩家都得全额分数
  - 人数不等 → **只有人数最多的玩家得分**

### 2. 小人管理

- 每人只有7个可用小人（记分板上那个不算）
- 小人用完只能等完成区域收回才能继续放人
- **权衡：** 快速完成小区域收回小人 vs 长期占领大区域

### 3. 时机选择

- 何时完成区域（提前收回小人 vs 继续扩大区域）
- 何时共享区域（合作得分 vs 增加人数抢夺）
- 何时放农民（长期投资 vs 短期灵活性）

### 4. 进阶要素

**农民规则：**
- 横躺在草地上，游戏结束时每个完成城市3分
- **一旦放置不能收回**，需承担长期占用小人的代价

**河流扩展：**
- 源头 → 10张河流拼图 → 湖泊 → 普通拼图
- 两张弯道不能连续往同一方向拐

**修道院长：**
- 可提前收回并按当前进度计分（如九宫格只拼8张，收回时得8分）
