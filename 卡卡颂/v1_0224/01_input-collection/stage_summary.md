## 🔄 后续阶段准备

### Step 2: 规则讲解图

**内容重点：**
- 回合流程（抽→放→人→分）- 30%
- 计分规则（道路/城市/修道院）- 40%
- 区域共享（人数多者得分）- 20%
- 小人管理（7个限制）- 10%

**视觉风格：** Notion风格信息图，橙白配色

### Step 3: 封面图生成

**三条风格线：**
1. **文艺调性线**（用户偏好）：南法小镇午后 + 拼图特写 + 手写标题
2. **高点击率线**：明亮色块 + 悬念文案 + 对比强烈
3. **人物产品线**：朋友围坐 + 游戏盒子 + 温馨氛围

### Step 4: 内页图生成

**4张内页规划：**
- P2 教学页：回合流程图解（抽→放→人→分）
- P3 解析页：计分规则对比（完成vs未完成）
- P4 场景页：游戏进行中的俯视桌面（地图拼接+小人占领）
- P5 技巧页：进阶策略（区域共享+农民规则）

### Step 5: 文案撰写

**标题公式：**
1. 热点适配：「周末在家，用72张拼图拼出一座南法小镇」
2. 情感共鸣：「20年前的德国桌游，治愈了我的周末焦虑」
3. 悬念好奇：「为什么这款"拼图游戏"能让人上瘾20年？」
4. 权威背书：「德国年度桌游｜卡卡颂，经典永不过时」

**正文结构：**
1. 场景开篇（午后/咖啡桌/朋友）
2. 游戏介绍（获奖/机制）
3. 核心玩法（用故事讲规则）
4. 策略乐趣（博弈/互动）
5. 适合场景（2人/多人/家庭）
6. 情感收尾（陪伴/推荐）

### Step 6: 发帖策略

**时间建议：**
- 周末前（周四晚/周五早）发布
- 节假日提前3天预热

**A/B测试方案：**
- A版：文艺调性封面 + 情感共鸣标题
- B版：高点击率封面 + 热点适配标题
