# 屁者先知规则

## META | 游戏元信息

| 字段 | 内容 |
|------|------|
| 游戏名称 | 屁者先知 |
| 游戏类型 | 身份推理 / 阵营对抗 / 卡牌 |
| 玩家人数 | 5~10人 |
| 建议人数 | 6~8人 |
| 游戏时长 | 约20~30分钟 |
| 适合年龄 | 10岁以上 |
| 一句话介绍 | 电梯里的"放屁版狼人杀"——屁者偷偷放屁，乘客合力揪出真凶 |
| 核心机制 | 隐藏身份、行动 programming、投票淘汰、伤害结算 |
| 游戏结构 | 多轮制（共8层电梯，每层一轮） |
| 主题背景 | 拥挤的电梯里突然传来异味，谁能找出那个偷偷放屁的人？ |

## COMPONENTS | 游戏组件

| 组件名称 | 数量 | 说明 |
|----------|------|------|
| 角色牌 | 65张（不重复） | 每位玩家选择一张，获得独特能力 |
| 身份牌 | 依人数设置 | 分为屁者阵营和乘客阵营 |
| 行动牌 | 每人4张 | 每回合打出一张，决定本回合行动 |
| 环境牌 | 8张 | 代表电梯8层楼的事件（有屁/无屁） |
| 血量卡 | 每人1张 | 记录玩家剩余血量 |
| 号码牌 | 每人1张 | 决定发言和结算顺序 |

### 行动牌类型（共6种）

| 牌名 | 效果简述 |
|------|----------|
| 压 | 若有屁回合打出，免疫本次所有伤害 |
| 骂 | 若有屁回合打出，抽一张牌且本回合投票权+0.5 |
| 吹 | 若有屁回合打出，对一位相邻玩家造成1点伤害 |
| 吸 | 若无屁回合打出，补充两张手牌；若有屁回合打出，额外受到1点伤害 |
| 抓 | 可在有屁回合的行动阶段直接打出，对任意玩家造成1点不可防御的伤害 |
| 听 | 若有屁回合打出，可秘密检视一位活跃玩家的身份牌 |

### 环境牌类型

| 牌型 | 效果 |
|------|------|
| 无屁 | 安全，无事发生 |
| 有屁 | 所有玩家受到1点伤害 |
| 臭屁 | 所有玩家受到2点伤害 |
| 闷屁 | 对所有压牌玩家造成1点伤害，对所有未压牌玩家造成3点伤害 |

## SETUP | 游戏准备

1. **分配身份**
   - 根据游戏人数，按比例发放身份牌（屁者 vs 乘客）
   - **5人局**：3乘客 + 2屁者
   - **6人局**：4乘客 + 2屁者
   - **7人局**：4乘客 + 3屁者
   - **8人局**：5乘客 + 3屁者
   - **9人局**：6乘客 + 3屁者
   - **10人局**：6乘客 + 4屁者
   - 每人抽取一张身份牌，**秘密查看后扣置**，不要让别人看到

2. **选择角色**
   - 每人从两张角色牌中选择一张保留，另一张弃置
   - 角色牌提供独特能力，公开或秘密依角色效果而定

3. **发放行动牌**
   - 每人获得4张行动牌作为起始手牌

4. **设置血量**
   - 每人拿取一张血量卡，初始血量为4点

5. **设置环境牌**
   - 将8张环境牌洗混，扣置形成环境牌堆
   - 环境牌代表电梯将经过的8层楼

6. **分配号码牌**
   - 每人随机抽取一张号码牌，决定发言和行动结算顺序

7. **屁者睁眼阶段**
   - 主持人宣布"天黑请闭眼"
   - 所有玩家闭眼
   - 主持人依次提示"屁者1号睁眼""屁者2号睁眼"...
   - 屁者可以查看全部8张环境牌的顺序，知晓哪一层有屁
   - **注意**：屁者互不知道彼此身份

## TURN_FLOW | 回合流程

游戏共进行8个回合，代表电梯经过8层楼。每回合流程如下：

### 步骤1: 压注行动牌

所有玩家同时选择一张行动牌，面朝下扣置在自己面前。

### 步骤2: 翻开环境牌

主持人翻开本回合的环境牌，宣布当前楼层事件：
- **无屁**：安全楼层
- **有屁/臭屁/闷屁**：危险楼层，将触发伤害

### 步骤3: 结算伤害

根据环境牌类型结算伤害：

| 环境牌 | 伤害效果 |
|--------|----------|
| 无屁 | 无人受伤 |
| 有屁 | 所有玩家受到1点伤害 |
| 臭屁 | 所有玩家受到2点伤害 |
| 闷屁 | 压牌玩家受1点伤害，未压牌玩家受3点伤害 |

**伤害减免**：若玩家本回合打出"压"且环境牌为有屁/臭屁/闷屁，则免疫本次所有伤害。

### 步骤4: 结算行动牌

按照号码牌顺序，依次翻开并执行行动牌效果：

| 行动牌 | 无屁回合效果 | 有屁回合效果 |
|--------|--------------|--------------|
| 压 | 无效果 | 免疫本次所有伤害 |
| 骂 | 无效果 | 抽1张牌，投票权+0.5 |
| 吹 | 无效果 | 对相邻玩家造成1点伤害 |
| 吸 | 补充2张手牌 | 额外受到1点伤害 |
| 抓 | 不可打出 | 直接对任意玩家造成1点不可防御的伤害 |
| 听 | 无效果 | 秘密检视一位活跃玩家的身份牌 |

### 步骤5: 投票环节（仅本回合有屁时触发）

若本回合环境牌为有屁/臭屁/闷屁，则进入投票环节：

1. **发言阶段**：按照号码顺序，每位玩家简短发言（可跳过）
2. **投票阶段**：所有玩家同时投票，选出最可疑的玩家
3. **出局结算**：
   - 得票最多的玩家出局，亮出身份牌
   - 若平票，则由主持人决定（或平票者均不出局，依房规）

### 步骤6: 检查终局条件

每回合结束后检查是否满足终局条件，若未满足则进入下一回合。

## SPECIAL_RULES | 特殊规则

### 角色能力（部分示例）

游戏包含65张不同的角色牌，部分角色能力示例：

| 角色名 | 能力效果 |
|--------|----------|
| 社恐 | 跳过发言阶段，但血量+1 |
| 臭嘴 | 可以将"吹"当作"臭屁"使用 |
| 忍者 | 免疫各种伤害 |
| 雷电法王 | 特殊攻击能力 |
| 奥本海默 | 特殊效果 |
| 匹诺曹 | 特殊效果 |
| 键盘侠 | 特殊效果 |

> 完整角色能力请查阅角色牌面说明。

### 血量归零

当玩家血量降至0或以下时，该玩家**立即出局**，亮出身份牌，但手牌不公开。

### 手牌管理

- 行动牌打出后进入弃牌堆
- 手牌无上限
- 当牌堆耗尽时，将弃牌堆洗匀形成新牌堆

## GAME_END | 游戏终局

### 终局触发条件（满足任一即结束）

1. **8层楼全部结束**：电梯到达顶层，游戏强制结束
2. **所有屁者被票出**：乘客阵营达成目标
3. **乘客存活数 ≤ 屁者存活数**：屁者阵营达成目标

### 最终胜利判定

| 条件 | 胜利方 |
|------|--------|
| 所有屁者被投票淘汰 | **乘客胜利** |
| 乘客存活人数 ≤ 屁者存活人数 | **屁者胜利** |
| 8层楼结束后，以上条件均未达成 | 按当前存活人数比较，人数多的一方获胜；若相同则屁者胜利 |

## VARIANTS | 变体规则

### 进阶规则：超自然角色

加入具有特殊能力的进阶身份牌，增加游戏复杂度：
- 部分角色拥有主动技能
- 部分角色拥有被动效果
- 建议熟悉基础规则后使用

### 快速模式

- 将8层楼缩短为5层
- 适合时间紧张时游玩

### 新手模式

- 移除"闷屁"环境牌
- 简化伤害计算

## GLOSSARY | 术语表

| 术语 | 定义 |
|------|------|
| 屁者 | 隐藏身份阵营，目标是让存活乘客数≤屁者数 |
| 乘客 | 公开身份阵营，目标是找出并淘汰所有屁者 |
| 环境牌 | 代表电梯楼层事件的牌，决定本回合是否有屁 |
| 行动牌 | 每回合打出的牌，决定玩家本回合的行动 |
| 压 | 行动牌类型，可免疫有屁回合的伤害 |
| 血量 | 玩家生命值，归零则出局 |
| 出局 | 玩家被淘汰，身份公开 |
| 相邻玩家 | 座位左右两边的玩家 |
